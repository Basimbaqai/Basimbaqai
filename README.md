## Work

| Project | Description | Stack |
|---|---|---|
| **Agentic pipeline framework** | Multi-step reasoning agent with tool use and persistent memory | LangChain · Claude API · FastAPI |
| **Real-time object detection** | YOLO-based pipeline with quantization and ONNX export for edge deployment | YOLOv8 · OpenCV · ONNX |
| **Domain-specific LLM fine-tuning** | SFT + DPO pipeline using QLoRA on a 7B base model, served with vLLM | QLoRA · HuggingFace · DPO · vLLM |
| **Facial recognition system** | Deep embedding pipeline with ArcFace loss and FAISS-backed search | ArcFace · OpenCV · FAISS |
| **Image classification (ViT)** | Transfer learning with fine-tuned Vision Transformer backbones | ViT · EfficientNet · PyTorch |
